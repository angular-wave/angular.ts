# Changelog

## 0.0.1

Initial unpublished AngularTS VS Code extension milestone.

- Added built-in AngularTS directive catalog, completion, hover, definition,
  and diagnostics.
- Added custom directive, component, filter, controller, and injectable
  indexing.
- Added component/directive binding completion, hover, definition, and usage
  search.
- Added filter completion, hover, definition, signatures, chained-filter
  parsing, and unknown-filter diagnostics.
- Added dependency-injection diagnostics and navigation for DI arrays and
  `$inject` assignments.
- Added controller, route/state component, template URL, and service
  navigation.
- Added component generation command and AngularTS snippets.
- Added root `make vscode-build` and `make vscode-test` entry points.
